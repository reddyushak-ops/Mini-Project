# Employee Leave Management System

A simple Flask-based web application for managing employee leave requests.

## Features
- Employee Leave Request
- Leave Tracking
- Simple User Interface

## Technologies Used
- Python
- Flask
- HTML
- CSS

## Project Structure

Mini-Project/
├── templates/
│   └── index.html
├── static/
│   └── style.css
├── app.py
├── requirements.txt
└── README.md

## Run the Project

pip install -r requirements.txt
python app.py
